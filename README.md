# ∏(tet)ris

Mathematical symbols (+, π, …) as blocks instead of the usual tetra-blocks

Special points for clearing each symbol

Connect controllers for more fun

---

## How to Play

Download the HTML file and simply open it!

---

## Troubleshooting

> *(made it quickly..)*

**Multiplayer mode:**
Refresh (`Command + R`), then change mode before starting the game.

**Block movements:**
Do not hold or long-press the buttons — press them multiple times instead.
*(the game doesn't detect long presses)*

**Controllers:**
If controllers do not work or show as disconnected, refresh and press one button.
*(successful if connection is shown)*

Sometimes, dual controllers may be connected to the other side in multiplayer mode. The game still works — consider it a challenge!
